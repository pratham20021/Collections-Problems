# Collections Problems

This repository contains Java programs demonstrating various operations and problems related to Java Collections Framework.

## Structure

- **src/List/**: Programs demonstrating List operations (ArrayList, LinkedList, etc.)
- **src/Map/**: Programs demonstrating Map operations (HashMap, TreeMap, etc.)
- **src/Set/**: Programs demonstrating Set operations (HashSet, TreeSet, etc.)
- **src/Queue/**: Programs demonstrating Queue operations

## Programs Included

### List Programs (12 programs)
- Prog_1.java to Prog_12.java - Various List operations and algorithms

### Map Programs (3 programs)
- Prog_1.java to Prog_3.java - Map operations and use cases

### Set Programs (1 program)
- Prog_1.java - Set operations

## How to Run

1. Clone the repository
2. Navigate to the src directory
3. Compile: `javac <package>/<ProgramName>.java`
4. Run: `java <package>.<ProgramName>`

## Topics Covered

- List operations (add, remove, search, sort)
- Map operations (put, get, iteration)
- Set operations (add, contains, intersection)
- Collection algorithms and utilities
